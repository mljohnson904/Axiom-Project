# Prospect Builder

## Purpose

A single-page mobile-first prospecting app for Axiom AI Systems. It helps track local service businesses, evaluate fit, save outreach notes, manage follow-ups, and export prospect data.

## Current Direction

Keep the tool simple, local, and practical:

- Single `index.html` app
- LocalStorage data persistence
- No external APIs
- No scraping
- No automated DMs
- Mobile-first dark Axiom style

## Active Improvements

- Follow-ups due today or overdue
- JSON backup and restore
- Canned note presets
- Daily prospecting target checklist
- Outreach history fields
- Follow-up message helper
